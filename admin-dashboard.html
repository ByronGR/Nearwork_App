Add client users
Each user gets a Firebase account with the role client and is linked to this organization. They'll receive an email with a password-reset link — that's how they set their own password and activate their account.
📧
What happens when you click Create:
Firebase creates each user account with a secure temporary password. We immediately call sendPasswordResetEmail() — Firebase sends them an email with a link to set their own password. Their account is inactive until they click that link.
